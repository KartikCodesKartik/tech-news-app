# Tech News App - Frontend

React-based frontend for the Tech News application built with Vite for fast development and optimized production builds.

## Features

- Modern React with hooks and Context API
- Fast development with Vite and Hot Module Replacement (HMR)
- React Router for client-side routing
- Axios for API communication
- Responsive design with custom CSS
- Authentication and authorization
- Admin dashboard and article management
- Public article browsing and newsletter subscription

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Configuration

Create a `.env` file based on `.env.example`:
```
VITE_API_URL=http://localhost:5000/api
```

See the main project README for complete setup instructions.
